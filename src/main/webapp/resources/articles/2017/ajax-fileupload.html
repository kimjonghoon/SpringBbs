Ajax 파일 업로드

예제 다운로드
(파일 업로드/다운로드/삭제 기능을 Rest example에 추가했음)

테스트

압축을 풀고, 루트 디렉터리로 이동하여 mvn jetty:run을 실행한 후, http://localhost:8080을 방문한다.

관련 글
  • Jetty Maven Plugin
  • 게시판 기본 기능
  • 답변형 게시판
  • 오라클 계층형 쿼리를 사용하는 게시판
  • 모델 1 게시판
  • 모델 2 게시판
  • DBC를 사용하는 오라클 용 모델 2 게시판
  • DBCP를 사용하는 MySQL 용 모델 2 게시판
  • REST 예제
