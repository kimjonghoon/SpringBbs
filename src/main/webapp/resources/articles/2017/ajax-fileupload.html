Ajax File Upload

동영상을 보면서 댓글을 작성/수정/삭제하는 예제에 파일 업로드/다운로드/삭제 기능을 추가한 예제

예제 다운로드

테스트

압축을 풀고 루트 디렉터리로 이동하여 mvn jetty:run 실행 후, http://localhost:8080을 방문한다.

관련 글
  • Jetty Maven Plugin
  • 게시판 기본 기능
  • 계층형 게시판
  • 오라클 계층형 쿼리를 사용하는 계층형 게시판
  • 모델 1 게시판
  • 모델 2 게시판
  • DBCP를 사용하는 오라클 용 모델 2 게시판
  • DBCP를 사용하는 MySQL 용 모델 2 게시판
  • REST 예제
  • 블로그 목록
