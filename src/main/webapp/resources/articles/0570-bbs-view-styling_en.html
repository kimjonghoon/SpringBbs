Create a JavaScript toggle function for modifying comments

Create a working JavaScript function by clicking on the Modify link.

If you click on the Modify link, a modification form containing the text area should appear.
(This modification form is hidden by default)
Paste the following JavaScript code before </ head>: 

<script type="text/javascript">
function modifyCommentToggle(articleNo) {
    var p_id = "comment" + articleNo;
    var p = document.getElementById(p_id);
    
    var form_id = "modifyCommentForm" + articleNo;
    var form = document.getElementById(form_id);
    
    var p_display;
    var form_display;
    
    if (p.style.display) {
            p_display = '';
            form_display = 'none';
    } else {
            p_display = 'none';
            form_display = '';
    }
    p.style.display = p_display;
    form.style.display = form_display;
}
</script>

Try to click the Modify and Cancel links.
Make sure the JavaScript function is working properly.

Modify and Cancel links work the same way.
If you click the link when the modify form is visible, the modify form appears.
If you click the link when the modify form is not visible, the modify form disappears.
The p element works exactly the opposite.
This is called a toggle.
